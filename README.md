# ComfyUI 图像等比倍数缩放插件

## 插件介绍

这是一个功能强大的ComfyUI图像缩放节点，支持按倍数等比放大或缩小图像，具有独立的放大/缩小开关控制，确保操作逻辑清晰，避免冲突。

## 主要特性

- ✅ **等比缩放** - 保持原始宽高比，图像不变形
- ✅ **独立开关** - 放大和缩小功能互斥，避免误操作
- ✅ **倍数控制** - 支持0.01-8.0倍缩放，步长0.01
- ✅ **智能处理** - 自动处理无效倍数，防止错误
- ✅ **高质量算法** - 使用LANCZOS重采样，保证图像质量
- ✅ **中文界面** - 全中文UI，使用更便捷
- ✅ **尺寸输出** - 同时输出图像和新尺寸信息
